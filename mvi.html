Архітектура MVI (Model-View-Intent)

Архітектура MVI (Model-View-Intent) - це шаблон проектування, який розділяє бізнес-логіку від інтерфейсу користувача. Це робиться для окремого управління різними аспектами додатка.

Структура MVI

  • Модель (Model): Модель представляє стан додатка. Вона відповідає за збереження, відновлення та обробку даних для додатка.
  • Представлення (View): Представлення відображає дані, тобто візуалізує модель даних для користувача.
  • Намір (Intent): Намір представляє дії користувача, які викликають зміни в моделі.

Переваги MVI

  • Поділ відповідальності: Кожен компонент в архітектурі MVI має свою власну відповідальність, що спрощує управління додатком.
  • Полегшена підтримка: Оскільки різні частини додатка розроблені незалежно одна від одної, це полегшує підтримку та оновлення додатка.
  • Гнучкість: Завдяки розділенню на модель, представлення та намір, розробники можуть працювати паралельно на різних частинах додатка без взаємного впливу.
            
            // State (Model)
            public class MyState {
                private String data;
    
                public String getData() {
                    return data;
                }
    
                public void setData(String data) {
                    this.data = data;
                }
            }
    
            // View
            public class MyView extends AppCompatActivity {
                private MyState state;
                private PublishSubject intentSubject;
    
                @Override
                protected void onCreate(Bundle savedInstanceState) {
                    super.onCreate(savedInstanceState);
                    setContentView(R.layout.my_activity);
    
                    state = new MyState();
                    intentSubject = PublishSubject.create();
    
                    // Subscribe to the intents
                    intentSubject.subscribe(this::handleIntent);
                }
    
                private void handleIntent(String intent) {
                    // Update the state based on the intent
                }
    
                public void userAction() {
                    // Emit an intent
                    intentSubject.onNext("userAction");
                }
            }
